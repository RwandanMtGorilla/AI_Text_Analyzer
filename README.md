AI Text Analyzer

AI Text Analyzer 是一个简单的前后端分离项目，用于通过OpenAI API分析文本。

特点
基于Flask的后端API
使用JQuery进行前端和后端的交互
前端提供简洁和用户友好的界面

部署与使用
1. 克隆仓库
2. 设置虚拟环境 (推荐)
    python3 -m venv venv
    source venv/bin/activate
    或（windows）
    python -m venv venv
    venv\Scripts\Activate
3. 安装依赖
    pip install -r requirements.txt
4. 启动后端服务器
    python main.py
此时，你的后端应该在http://localhost:5000/运行。
5. 打开前端页面
使用浏览器打开 index.html 文件。
